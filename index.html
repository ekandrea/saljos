🚀 Hur du driftsätter Säljös på en riktig URL

1
Ladda ned ZIP-filen (du har redan gjort detta) och packa upp den.
2
Skapa ett gratis Vercel-konto på vercel.com (GitHub-login funkar direkt).
3
Dra och släpp mappen på vercel.com/new → välj "Import local project". Du får en URL direkt, t.ex. saljos.vercel.app
4
Lägg till environment variables i Vercel → ditt projekt → Settings → Environment Variables:
# Krävs för cloud sync:
JSONBIN_MASTER_KEY = din_jsonbin_master_key
JSONBIN_BIN_ID = ditt_bin_id
# Krävs för Bolagsverket:
BV_API_KEY = din_bolagsverket_nyckel
# Valfritt (om Bolagsverket ändrar URL):
BV_BASE_URL = https://api.bolagsverket.se/
5
Klicka Redeploy i Vercel efter att du lagt till variablerna. Klart! Allt fungerar nu.
💡 Tips: Registrera gratis på jsonbin.io → skapa ett bin → kopiera Master Key och Bin ID.

🔌 API-status

Klicka "Kontrollera alla" för att se vilka API:er som är aktiva.

💾 Datahantering

Data sparas i molnet (JSONBin) via /api/sync — rensa webbläsaren hur du vill.
